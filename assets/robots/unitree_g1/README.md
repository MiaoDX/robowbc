# Unitree G1 MuJoCo Model

The `g1_29dof.xml` MJCF model is sourced from
[unitree_mujoco](https://github.com/unitreerobotics/unitree_mujoco)
(`unitree_robots/g1/g1_29dof.xml`).

Mesh STL files are not included in this repository. Only the MJCF XML is
used for kinematic chain description, joint definitions, and limit
verification.
